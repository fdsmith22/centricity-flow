Application Initialization Flow
User navigates to app URL
Nginx serves React app
React app initializes providers
Check authentication status
Redirect to login if needed
Fetch user permissions
Initialize MobX stores
Establish WebSocket connections
Load configuration data
Render UI with permissions
WebSocket Interactions
Configuration Management WebSocket
Purpose: Real-time configuration synchronization
Messages: 150+ entity CRUD operations
Flow: Bidirectional configuration changes
Auth: dashboard_config_management_access
Command Management WebSocket
Purpose: Asynchronous command execution
Messages: CommandGroupRequest, CommandRequest
Flow: Request/Response with correlation IDs
Auth: centricity_command_management_access
Pipeline Reconfiguration WebSocket
Purpose: Dynamic pipeline updates
Messages: ReconfigurePipeline, PipelineReconfigured
Flow: Request/Response pattern
Auth: centricity_pipeline_reconfiguration_access
Controller State WebSocket
Purpose: Monitor controller states
Messages: Controller state updates
Flow: Server-to-client push
Auth: dashboard_controller_state_access
